# Clypra 简体中文本地化设计

日期：2026-07-21  
基线版本：Clypra v1.1.1（提交 `e5e2cbf`）

## 目标

为 Clypra v1.1.1 制作默认使用简体中文的 macOS 桌面版本，覆盖用户可见的本地界面文字，同时保留现有项目和设置数据，并保留可恢复的英文原版备份。

## 非目标

- 本次不提供语言切换设置。
- 不翻译文件名、用户输入、快捷键按键、编解码器名称、文件格式、品牌名和 API 返回的未知在线内容。
- 不重构与本地化无关的编辑、渲染、导出或媒体处理逻辑。
- 不改变项目文件格式、应用 bundle identifier 或数据目录。

## 当前状态

Clypra v1.1.1 没有国际化框架。界面文字直接分布在 React/TypeScript 组件内，初步扫描发现至少 368 个直接文本节点，另有标题、占位符、工具提示、错误信息和运行状态文字。应用的字幕语言设置仅控制 Whisper 转录语言，不是界面语言。

## 方案

采用集中式简体中文词典和统一翻译函数，不在组件中直接硬编码中文。

### 翻译模块

新增独立的本地化目录，包含：

- 稳定、按功能分组的翻译键。
- `zh-CN` 简体中文词典。
- 与中文词典键集合一致的英文回退词典。
- `t(key, params)` 翻译函数，支持数字、名称和状态等参数插值。
- 开发环境下的缺失键警告。

翻译函数保持无状态，应用启动后固定使用 `zh-CN`，不增加新的全局状态或持久化设置。

### 覆盖范围

汉化以下用户可见区域：

- 启动页、项目创建、最近项目和项目关闭流程。
- 编辑器导航、素材面板、时间线、轨道和播放控制。
- 文本、字幕、音频、滤镜、转场、贴纸和效果面板。
- 属性面板、画面比例、速度、音量、动画和模板编辑。
- 设置、快捷键说明、Whisper 模型和语言设置。
- 导出、录屏、更新、下载进度、崩溃恢复和网络状态。
- 模态框、按钮、空状态、占位符、工具提示、确认提示和面向用户的错误信息。

### 术语规则

- 使用中国大陆常见视频编辑术语，例如“时间线”“轨道”“转场”“关键帧”“画面比例”“导出”。
- H.264、H.265、ProRes、FPS、GPU、FFmpeg、Whisper、WebM 等标准名称保持原样。
- `⌘`、`Shift`、`Option` 等快捷键标记保持原样，只翻译功能说明。
- 用户文件名、项目名、模板自定义名称和媒体元数据保持原文。
- 在线目录返回的未知素材名或效果名使用英文回退，不显示空字符串。
- 中文文案优先简洁，避免因长度增加破坏现有布局。

## 数据流

组件通过翻译键调用 `t()`。翻译模块读取 `zh-CN` 词典，完成参数插值后返回字符串。如果键缺失，则返回英文回退并在开发环境记录警告；若英文回退也缺失，则返回可识别的键名，避免界面空白。

本地化层只处理显示文字，不修改项目状态、时间线数据、媒体路径、导出配置或后端 Tauri 命令参数。

## 错误处理

- 缺少中文翻译：使用英文回退并输出开发警告。
- 参数缺失：保留明确的占位标记并输出开发警告，便于测试发现。
- 在线内容无翻译：原样显示服务端内容。
- 构建或启动失败：不替换现有 `/Applications/Clypra.app`。
- 安装后验证失败：恢复英文原版备份。

## 测试与验证

### 自动检查

- 为 `t()` 添加键查找、参数插值和回退行为测试。
- 校验中文词典与英文回退词典的键集合一致。
- 扫描生产组件中的直接英文界面文字，允许明确列入白名单的标准术语和技术标识。
- 运行现有 TypeScript、单元测试和生产构建。

### 人工冒烟测试

启动构建后的 macOS 应用并检查：

1. 启动页与创建项目。
2. 素材导入和编辑器主界面。
3. 时间线、属性面板和播放控制。
4. 文本、字幕、滤镜、转场、贴纸和音频标签页。
5. 设置、导出、录屏、更新和恢复弹窗。
6. 长中文文本是否溢出、截断或遮挡控件。
7. 现有项目能否正常打开，数据路径是否保持不变。

## 构建与安装

1. 从 v1.1.1 源码构建 macOS Apple Silicon 版本。
2. 验证构建产物可启动后，再处理现有安装。
3. 退出正在运行的 Clypra。
4. 将英文原版保存为 `/Applications/Clypra-English-1.1.1.app.backup`，避免 LaunchServices 将备份误识别为另一份可启动应用。
5. 将中文版安装为 `/Applications/Clypra.app`，保持产品名、bundle identifier 和数据目录不变。
6. 启动中文版并执行冒烟测试。
7. 如果验证失败，移除失败构建并恢复英文原版。

## 完成标准

- 所有本地、用户可见的静态界面选项均显示为简体中文。
- 未翻译的动态在线内容可以安全回退英文。
- 应用成功构建并在当前 Mac 上启动。
- 现有项目和设置可继续使用。
- 自动检查未发现未获准的直接英文界面文字。
- 英文原版可恢复，安装失败不会导致用户失去可用应用。
